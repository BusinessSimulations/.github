# .github

Organization-wide community health files for the BSL organization.

GitHub automatically applies files in this repository to all organization repositories, unless they define their own versions.

## Contents

### Issue Templates

Located in `.github/ISSUE_TEMPLATE/`:

- **bug.yml** - For reporting bugs (applies `bug` label)
- **feature.yml** - For feature requests (applies `enhancement` label)
- **maintenance.yml** - For maintenance work (applies `maintenance` label)

These templates appear as options when creating a new issue in any repository.
